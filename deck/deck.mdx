### A complete end-to-end testing experience.

Cypress enables you to write all types of tests:

* End-to-end tests
* Integration tests
* Unit test

Cypress can test anything that runs in a browser.

---

## Cypress is not Selenium

Pros
* Easier onboarding
* Direct DOM manipulation
* Easy to debug tests

Cons
* Only supports JS
* Chrome only
